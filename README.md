## 📱 Responsive Design Details

### Typography Scaling
```css
/* Large Desktop */
.hero h1 { font-size: 5rem; }

/* Desktop */
.hero h1 { font-size: 4rem; }

/* Tablet */
.hero h1 { font-size: 3rem; }

/* Mobile */
.hero h1 { font-size: 2rem; }
```

### Grid Layouts
```css
/* Desktop */
.projects-grid {
    grid-template-columns: repeat(auto-fit, minmax(300px, 1fr));
}

.projects-grid {
    grid-template-columns: 1fr;
}
```
